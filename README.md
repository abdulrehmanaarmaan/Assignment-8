App Name: Hero Apps

Description: Hero Apps is a React-based web application that allows users to explore a variety of applications, view detailed information about each one and install them with a single click. It's designed with a clean, gradient-based UI and smooth navigation transitions to deliver an app-store-like experience right inside the browser.

Technologies: React.js, React Router, Tailwind CSS, Daisy UI, React Loader Spinner, React Icons, Local Storage API, Recharts
